A simple plugin that notifies customers if they already own the product they are viewing.

== Description ==

The 'Product Ownership Alert for WooCommerce' plugin helps WooCommerce store owners avoid redundant purchases. 
When a logged-in customer visits a product they've already purchased, it displays a customizable message will redirect them to the order page where they still have download permissions. 

== Frequently Asked Questions ==

= Can I change the notice text? =

Yes, you can customize both the notification prefix and the order page text by navigating to WooCommerce → Settings → Products → Downloadable Products.

= Is the plugin compatible with variable downloadable products? =

As of now, the plugin exclusively supports simple products.
